Tutorial 4: Working with Large Datasets
=========================================

.. admonition:: What you'll learn
   :class: tip

   Master techniques for efficiently processing large seismic volumes that don't fit in memory.

**Estimated time**: 30-45 minutes

Please see the :doc:`../USER-GUIDE` for complete large dataset examples.

This tutorial covers:

- Lazy loading strategies
- Chunked processing with Dask
- Memory management
- Progress monitoring
- Performance optimization
