Tutorial 2: Visualizing Seismic Data
======================================

.. admonition:: What you'll learn
   :class: tip

   In this tutorial, you'll learn how to create professional seismic visualizations including inline/crossline sections, time slices, and attribute maps.

**Estimated time**: 30-40 minutes

Please see the :doc:`../USER-GUIDE` for complete visualization examples.

This tutorial covers:

- Creating seismic section plots
- Plotting time slices
- Multi-panel visualizations
- Customizing colormaps
- Creating attribute maps
